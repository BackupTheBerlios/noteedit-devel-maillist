the noteedit release 2.7.4 is now online. Please check it out and enjoy. 
Thanks everybody who contributed and made it possible! Svn is now widely 
open for improvements again.

Okay, I messed a bit around 'till I got the tarball up, even had to take 
my first one down again, so here is shown what's finally inside, also 
for future reference (it's a makefile extract):

B = noteedit-2.7.4
A = noteedit-2.7.4_beta1

build_tar_ball:
    rm -rf $B
    svn export svn+ssh://svn.berlios.de/svnroot/repos/noteedit $B
    cp $A/configure.in         $B
    cp $A/Makefile.in          $B
    cp $A/acinclude.m4       $B
    cp $A/aclocal.m4            $B
    cp $A/configure              $B
    rm -f $B/Makefile.cvs
    tar cvf $B.tar $B
    gzip $B.tar

You see, I had a beta tarball around. I wonder only where the beta got 
these files from :)
For changes see the NEWS file and the website. Don't know how to tag svn 
version 60 with this release event.
