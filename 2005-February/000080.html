Currently only me and Georg can do this.
I go to Admin -> Edit/Release files
I upload the tarball to ftp.berlios.de

After that I can start to create a new release.
You either have to be admin or release technician to do the job.
I had planned to do it myself.

I'd used the current source from SVN, create a release TAG to it
and add the old configure / admin files to the files and remove
the CVS specific files.

I change your user data.

Core 3 for myself (tarball and rpms).
