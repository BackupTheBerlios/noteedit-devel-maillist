Hi,

   I had this problem too but I solved it with this patch. I made it
after a patch that was in the patches section from the project site.
It seems that qt 3.3.5 doesen't have the nscaledit.h file anymore (my
guess...) and there was two files in the Noteedit distribution that
refered to it. One file is noteedit/staffProps.ui and the other is
noteedit/pmxexportform.ui. The patch in the project site
(noteedit-2.8.0-qt-3.3.5.patch) was merged into the CVS for the beta
release. But the second file still suffer from this problem and this
patch can solve it. I applied it with

patch -Np1 -i noteedit-2.8.1-qt-3.3.5.patch

in the top level dir of the Noteedit distribution.

I hope that helps.
