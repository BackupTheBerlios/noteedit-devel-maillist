Hi,

If anyone wants to start reducing some warnings in NoteEdit he can do it
in mainframewidget.cpp:

Replace the keyboard shortcut creation insertItem() and connectItem() by
one single line insert(). The insert method requires an What's this 
string (i.e. a more or less detailed help, more than a simple tool tip).
I've added a patch to this mail not yet in svn so you get an idea how
to it.
