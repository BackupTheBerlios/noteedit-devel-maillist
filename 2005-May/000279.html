Hi,

I've found know way to remove the std-menu item
"Report bug".

I tried two ways to get rid of it (the first one is commented):

	// Get report bug action to remove it from the help menu
	//KAction *rbug = 
actionCollection()->action(KStdAction::stdName(KStdAction::ReportBug));
	KAction *rbug = KStdAction::reportBug( this, SLOT(fileSaveAs()), 
actionCollection());
	if( rbug )
	{
	  rbug->unplugAll();
	  //actionCollection()->remove(rbug);
	}
	else
	  printf("Could not remove ReportBug menu\n");

When trying to access the menu using actionCollection() the rbug pointer
is uninitialized. It seems it gets created after the constructor. This
might be the reason why the second method fails as well.

unplugall removes the action from all toolbars and menus according to
the KDE documentation. remove does remove it from the action list.

Any ideas are welcome how to solve the problem.
