In other words: I discovered that triplet input works only when you 
insert notes with the mouse, not with the midi keyboard. This is in non 
edit mode, of course. If you want to see it: start noteedit, kaconnect 
and vkeybd. Connect in kaconnect vkeybd (in my setup usually 131:0) with 
noteedit (usually 130:0). When you press the black keyboard icon in 
noteedit you should hear the playing from vkeybd. When you press the red 
keyboard icon in noteedit you can enter notes like you do with the mouse.
