Reinhard,

1) I see you managed to release 2.8.0 these days. I would like to
remind you to make a copy of the released 2.8.0 version into
/tags/2.8.0 in our SVN repository.
2) I played downloaded&compiled&installed Qt4.0. I like it so far, but
wasn't able to make much with NoteEdit yet (configure doesn't find
it?!?). Can you say anything about your plans, when and how are you
going to migrate to Qt4.0? We'd have to update our configure to work
properly (or maybe even rewrite it completely?), remove code in #if
QT_VERSION <300 block, make changes to fit a new Qt API and announce
that NoteEdit 2.9 requires now Qt >= 300.
3) I tried compiling NoteEdit with gcc-4.0, but had no luck. I fixed
the configure script and it works fine. The problem is that it cannot
compile TSE3 headers (MANY syntax errors like the wrong use of
type-def, friend classes, templates etc.!). I contacted Pete and I
hope he'll manage to release a gcc4-compatible version asap. I don't
think Debian is going to accept our program without being gcc4-compatible.
