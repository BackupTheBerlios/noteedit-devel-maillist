Hi Csabe,

I had success with Midi all the time with the
following kernel versions on Mandrake/Mandriva:
2.6.8.1, 2.6.11, 2.6.12 (and also with the
alsa cvs snapshort I downloaded on 30. Juli).

You have to load the following midi modules:
snd-seq
snd-seq-virmidi
snd-seq-dummy ports=3
snd-seq-instr
snd-seq-midi
snd-seq-midi-emul

You should remove all midi modules (like snd-rawmidi)
activating your sound card midi hardware which might
be to root source for the freeze. I could imagine that
there are similar sound card types, one supporting
midi and the other one not. Also you should try the
latest stable alsa driver.

realtime issues only concern the audio output (using
jack) but not Midi itself.

Regards,

Reinhard
