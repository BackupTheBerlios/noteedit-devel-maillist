Hi Taco.

Some windows are already built with Qt designer (eg. Export file dialog,
Staff properties dialog and some other, smaller dialogs). However, the
main window is unfortunately not the case - the whole form is built in
the mainframe constructor.

About the crash, can you send a crash report (compile the project in
debugging mode and send the name of the function, where it crashes). If
it crashes when splash->show() method is called, try installing NoteEdit
by "make install".

You should also try to grab the latest SVN version of NoteEdit IMO, if
you're planning to do any changes on the code.
