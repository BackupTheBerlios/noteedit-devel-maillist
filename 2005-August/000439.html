Greetings to all!

I changed some default most commonly used keys in order to reunite the
UI with other apps as close as possible and make them more logically
placed. The following changes are:
End -> E: Edit mode
Alt-Right, Alt-Left -> Home, End: Move at the beginning of the document,
move at the end of document
Insert -> SHIFT+Return: Insert rest
CTRL+Backspace -> CTRL+Delete: Delete a note in chord
E -> SHIFT+E: Pitch E (a simple E key is now used for Edit mode and
Shift+E is the best suitable option for this IMO)

I also added a new ESC key, which immediately leaves the current mode
you're in. In the future, the second hit should unselect the current
selection as well and go to A voice.

Tell me what you think!
