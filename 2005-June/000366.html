Hi,

I have created a Flyer and a T-Shirt for NoteEdit / LAD / Linuxtag.
If you want to print your own one (there are dozens of shops printing
T-Shirts from PDF-Files) you have to change the Name on the back
of the T-Shirt. Alternatively you could change some of the Logos
(replacing them f.e. with the NoteEdit-Logo). Also everything
is black-white. (for dark-color-shirts you probably would have
to change the file as well).

The flyer is in available in english and german while the t-shirt
is only german (just kidding ;-). I did not include postscript or
pdf files but I recommand to print to postscript with OpenOffice
first as the pdf version of my OOWriter inverses the NoteEdit Logo.
To create a PDF you could use ps2pdf.
