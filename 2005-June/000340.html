I would like to report that the stem direction gets only remembered when
you open the voice dialog at least two times. It's reproducable:

    -- start new File
    -- create two voices, voice one stems up, voice two stems down
    -- add clef
    -- add time signature
    -- select 1st voice
    -- insert notes                    stems are up or down: not working,

but with this one:

    -- start new File
    -- create two voices, voice one stems up, voice two stems down
    -- add clef
    -- add time signature
    -- open voice dialog again, close it with ok
    -- select 1st voice
    -- insert notes                    stems are up: working.
