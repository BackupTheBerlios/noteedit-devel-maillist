Hi,

I started the port of the main window to raw Qt. I looked
into mainframewidget.cpp and found out that the code base
doesn't seem to be too difficult to move back for these
components. I includeded a designer ui file for a first
impression. I used the KDE crystalsvg icons as a base,
this can of course be changed if you know similar good
looking icons.

I also made several small corrections to the menus and
started grouping some of the insert of notes and other
musical symbols. There is a technical problem to this
though as you know longer see that you have selected
one of the symbols of such a group to be inserted (i.e.
you see which symbol is selected but not that you
have selected the group for input). This probably can
be worked around as a Toolbar is a Dockwindow where
other widgets could be added as well.

Of course several toolbars are still missing and some
menu points have to be added (Qt requires for every
toolbar item one menu item), and of course all the slots
for the menu have to be added to the mainwindow.ui.h file.
