Hi,

This evening I looked into the GUI patch. In contains the following 
changes to make the menu structure more logical:

- "New window" moved to the file menu (probably better to be placed
   below "New"
- Submenu for import and export
- First Import submenu, then Export submenu
- Close All (close all windows) in the file menu
- Comments changed in the edit menu (with errors)
- Setting (staff) moved to the menu Format
- New menu "Format"
- Extras/Tools menu moved to the right of the TSE3 menu
- Output parameters moved to the menu settings
- Swap "Keyboard config" and "Configure noteedit"
- Comment to the help menu (to be corrected).

To test the patch: Save the raw patch as noteeditui.rc and replace
this file from the one in the noteedit subdirectory. If you have
noteedit already installed you might need to replace the installed
file (restoring it should be no problem) as noteedit might use the
installed one instead of the one installed in the noteedit directory.

I'd like to have some feedback before I apply the (modified) patch
to svn. Thanks for your help!
